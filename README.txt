Transaction API - Test Guide
This document provides step-by-step instructions for setting up and testing the Transaction API.

1. Running the API
Open the API solution:
C:\Users\CMA SOLUTIONS\Downloads\Fortitude.Asia_Test\TransactionApi\TransactionApi.sln
Start the API using Visual Studio or the .NET CLI.

2. Running Unit Tests
Open Command Prompt (CMD).

Navigate to the unit test project directory:
cd C:\Users\xxxxx\Downloads\Fortitude.Asia_Test\TransactionApi\TransactionApi.Tests

List all available unit tests:
dotnet test --list-tests

Run unit tests with detailed logging:
dotnet test --logger "console;verbosity=detailed"

3. Log File Location
Test execution logs can be found at:
